# BigQuery

Chaos Genius uses [Airbyte's BigQuery connector](https://docs.airbyte.io/integrations/sources/bigquery) to sync data.

To connect BigQuery as a data source, you will need the following information:

-   Credentials JSON

-   Default Dataset ID

-   Project ID
